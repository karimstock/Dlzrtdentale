Comment connecter votre boite mail ?
Gmail
  1. Connectez-vous sur myaccount.google.com
  2. Allez dans Securite
  3. Activez la Validation en 2 etapes si pas deja fait
  4. Cherchez Mots de passe des applications
  5. Selectionnez "Autre (nom personnalise)" et tapez "JADOMI"
  6. Copiez le mot de passe de 16 caracteres
  7. Utilisez ce mot de passe dans JADOMI
Outlook / Hotmail
  1. Connectez-vous sur account.microsoft.com
  2. Allez dans Securite
  3. Cliquez sur Options de securite avancees
  4. Section Mots de passe d'application puis Creer
  5. Nommez-le "JADOMI" et copiez le mot de passe genere
  6. Utilisez ce mot de passe dans JADOMI
Si l'option n'apparait pas : activez d'abord la verification en 2 etapes
Yahoo Mail
🟣 Se connecter avec Yahoo
Ou utilisez un mot de passe d'application IMAP :
  1. Activez d'abord l'IMAP : Yahoo Mail > Parametres > Autres applis > Activer IMAP
  2. Connectez-vous sur login.yahoo.com
  3. Cliquez sur votre avatar puis Informations du compte
  4. Allez dans Securite du compte
  5. Cliquez sur Generer un mot de passe d'application
  6. Selectionnez "Autre application" et tapez "JADOMI"
  7. Copiez le code de 16 caracteres genere
  8. Utilisez ce code dans JADOMI
OVH / Orange / Free / SFR / Laposte
  1. Utilisez directement votre mot de passe habituel
  2. Ces fournisseurs ne necessitent pas de mot de passe d'application
  3. Si erreur : verifiez que l'IMAP est active dans les parametres de votre messagerie
